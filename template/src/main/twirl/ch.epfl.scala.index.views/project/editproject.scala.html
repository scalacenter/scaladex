Latest Version

When possible, we do not use release candidate or milestone releases as the last version. For example, if version 1.2.1 and version 2.0.0-RC1 of an artifact are released, we choose 1.2.1 as the latest version. If you would prefer we consider pre-release versions, such as 2.0.0-RC1, the latest version for this project's artifacts, disable this setting:
